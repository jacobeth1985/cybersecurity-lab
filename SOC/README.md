# Security Operations Center (SOC) Lab

Welcome to my Security Operations Center (SOC) Lab.

This folder documents my hands-on SOC projects, investigations, detection engineering, threat hunting activities, and incident response exercises as I develop practical cybersecurity skills.

The purpose of this lab is to simulate the daily responsibilities of a Security Operations Center (SOC) Analyst by working through realistic security scenarios, analyzing logs, investigating alerts, and documenting findings.
