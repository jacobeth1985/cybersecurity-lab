# Active Directory Lab

Welcome to my Active Directory Lab.

This folder documents my hands-on experience with Microsoft Active Directory, including user and group management, authentication, security, administration, and attack and defense techniques in a lab environment.

---

# About This Lab

Active Directory (AD) is Microsoft's directory service used to manage users, computers, groups, and resources within Windows enterprise environments. Understanding Active Directory is essential for SOC Analysts, Blue Team professionals, Incident Responders, System Administrators, and Penetration Testers.
